# Fortnite-Itemshop-by-Slone
Fortnite Item Shop.
The API used is Fortnite-API.com
https://fortnite-api.com

# How to use
1.Install the following libraries
```
pip install Pillow
pip install opencv-python
```
2.Execute run.bat
